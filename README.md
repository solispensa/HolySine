## Features
- **Precision Tracking**: Uses Autocorrelation (SDF) and FFT for extreme sub-cent accuracy.
- **Polyphonic Mode**: Tune all 6 strings at once.
- **Custom Tunings**: Add your own string layouts instantly.
- **Spectrogram**: Real-time FFT visualizer overlay.
- **Zero Dependencies**: Standalone HTML file you can run offline.

## Usage
Simply open `standalone.html` in any browser, or visit the GitHub Pages [live site](https://solispensa.github.io/HolySine/).
